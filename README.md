
# sarcasm
Just a small program that will upper-case every second letter.

## Features
1) Blazing(420)ly fast because it's written in C++. 
1) It checks for numbers and skips the upper-case check for those.
1) It checks for already upeer-case letters and skips the check for those aswell so that words/acronyms that need to stay upper-case, actually stay upper-case (for example: NASA)

## Compilation
Use any C++ compiler you have on your PC.

Linux:<br>
$ g++ sarcasm.cpp -o sarcasm
